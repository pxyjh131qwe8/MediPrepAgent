# 通用急诊危险信号

## 知识边界
本文档用于 MediPrepAgent 的 RAG 医疗科普知识库，只提供健康科普、危险信号提示、就诊准备和科室导航信息。不能作为疾病确诊、治疗方案或用药依据。

## 概述
通用急诊危险信号用于 MediPrepAgent 的安全兜底规则。当用户描述高危信号时，应优先建议急诊评估。

## 常见相关因素
- 胸痛、呼吸困难、意识模糊。
- 大量出血、昏迷、抽搐。
- 突发神经系统症状或严重疼痛。

## 需要及时就医或急诊评估的信号
- 突发剧烈胸痛或头痛。
- 呼吸困难、晕厥、意识异常。
- 肢体无力、说话不清、口角歪斜。
- 呕血、黑便、大量便血或外伤大量出血。

## 推荐就诊科室
出现通用急诊危险信号时，优先急诊科。

## 就诊前准备
- 记录“通用急诊危险信号”相关症状出现的时间、持续多久、频率、诱发因素和缓解因素。
- 记录是否伴随发热、胸痛、呼吸困难、意识异常、呕吐、出血、体重下降等情况。
- 整理既往病史、过敏史、长期用药、近期检查报告和体检结果。
- 若症状明显加重或出现危险信号，应及时线下就医。

## 日常注意与自我观察
- 不要等待 Agent 继续追问。
- 提醒联系当地急救或尽快到急诊。
- 记录症状开始时间和关键伴随症状。

## RAG 检索关键词
急诊, 危险信号, 胸痛, 呼吸困难, 意识模糊, 大量出血

## Agent 使用提示
生成回答时应使用“可能、相关、建议评估”等表述，不得输出“你已经确诊为某疾病”。如果命中高危信号，应优先提醒线下就医或急诊评估。

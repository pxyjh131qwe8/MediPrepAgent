# 压力管理

## 知识边界
本文档用于 MediPrepAgent 的 RAG 医疗科普知识库，只提供健康科普、危险信号提示、就诊准备和科室导航信息。不能作为疾病确诊、治疗方案或用药依据。

## 概述
压力会影响睡眠、情绪、注意力和身体感受。压力管理不等于忽视症状，而是帮助用户更好地观察和应对。

## 常见相关因素
- 工作学习负荷过重。
- 睡眠不足和缺乏运动。
- 缺少社交支持或长期紧张。

## 需要及时就医或急诊评估的信号
- 压力伴持续失眠、明显焦虑或抑郁。
- 出现自伤想法或无法维持日常功能。
- 伴胸痛、呼吸困难、晕厥等身体危险信号。

## 推荐就诊科室
轻度压力可自我管理；持续影响生活可考虑精神心理科或全科医学科。

## 就诊前准备
- 记录“压力管理”相关症状出现的时间、持续多久、频率、诱发因素和缓解因素。
- 记录是否伴随发热、胸痛、呼吸困难、意识异常、呕吐、出血、体重下降等情况。
- 整理既往病史、过敏史、长期用药、近期检查报告和体检结果。
- 若症状明显加重或出现危险信号，应及时线下就医。

## 日常注意与自我观察
- 建立固定作息和运动计划。
- 使用呼吸放松、正念练习或写日记。
- 必要时寻求心理咨询或医生帮助。

# 尿常规基础解读

## 知识边界
本文档用于 MediPrepAgent 的 RAG 医疗科普知识库，只提供健康科普、危险信号提示、就诊准备和科室导航信息。不能作为疾病确诊、治疗方案或用药依据。

## 概述
尿常规可提供泌尿系统感染、肾脏问题、出血线索等信息，但需要结合症状、复查和其他检查。

## 常见相关因素
- 尿白细胞可能与泌尿感染相关。
- 尿蛋白可能与肾脏或一过性因素相关。
- 尿潜血需结合女性经期、运动、结石等因素。

## 需要及时就医或急诊评估的信号
- 尿痛尿频伴发热、腰痛。
- 肉眼血尿。
- 尿量减少、水肿或血压明显升高。

## 推荐就诊科室
尿检异常可先全科；尿蛋白明显或肾功能异常可考虑肾内科；血尿或结石相关可考虑泌尿外科。

## 就诊前准备
- 记录“尿常规基础解读”相关症状出现的时间、持续多久、频率、诱发因素和缓解因素。
- 记录是否伴随发热、胸痛、呼吸困难、意识异常、呕吐、出血、体重下降等情况。
- 整理既往病史、过敏史、长期用药、近期检查报告和体检结果。
- 若症状明显加重或出现危险信号，应及时线下就医。

## 日常注意与自我观察
- 记录尿频、尿急、尿痛、腰痛和发热。
- 注意留取中段尿并避开污染因素。
- 携带肾功能、尿检复查结果。

## Agent 使用提示
生成回答时应使用“可能、相关、建议评估”等表述，不得输出“你已经确诊为某疾病”。如果命中高危信号，应优先提醒线下就医或急诊评估。

# 肝功能基础解读

## 知识边界
本文档用于 MediPrepAgent 的 RAG 医疗科普知识库，只提供健康科普、危险信号提示、就诊准备和科室导航信息。不能作为疾病确诊、治疗方案或用药依据。

## 概述
肝功能检查常见指标包括转氨酶、胆红素、白蛋白等。异常可能与肝胆疾病、药物、饮酒、脂肪肝等相关。

## 常见相关因素
- 饮酒、脂肪肝、病毒性肝炎、药物影响。
- 胆道问题可能影响胆红素。
- 剧烈运动也可能影响部分指标。

## 需要及时就医或急诊评估的信号
- 皮肤眼睛发黄、尿色加深。
- 明显乏力、食欲差、右上腹痛。
- 指标明显升高或伴凝血异常。

## 推荐就诊科室
肝功能异常可考虑消化内科或肝病科；轻度异常也可先全科评估。

## 就诊前准备
- 记录“肝功能基础解读”相关症状出现的时间、持续多久、频率、诱发因素和缓解因素。
- 记录是否伴随发热、胸痛、呼吸困难、意识异常、呕吐、出血、体重下降等情况。
- 整理既往病史、过敏史、长期用药、近期检查报告和体检结果。
- 若症状明显加重或出现危险信号，应及时线下就医。

## 日常注意与自我观察
- 携带历史肝功能对比结果。
- 说明饮酒、用药、保健品和乙肝等病史。
- 不自行服用护肝药。
